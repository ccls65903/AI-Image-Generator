# AI Image Generator

An AI-powered image generation application built using the MERN (MongoDB, Express.js, React.js, Node.js) stack.

## Features

- **AI-Powered Image Generation**: Leverages advanced AI models to create images based on user input.
- **User-Friendly Interface**: Intuitive design for seamless interaction and image creation.
- **Responsive Design**: Optimized for various devices, ensuring accessibility across platforms.

## Prerequisites

Before setting up the project, ensure you have the following installed:

- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)

## Usage

1. **Start the Development Server**:

   - In the root directory:

     ```bash
     npm run server
     ```

   - In the `client` directory:

     ```bash
     npm start
     ```

2. **Access the Application**:

   Open your browser and navigate to `http://localhost:3000` to use the application.
